# Contributing to Garaad

Thank you for your interest in contributing to Garaad! We are thrilled to have you join our community.

## How to Contribute

### Reporting Bugs
If you find a bug, please create an Issue detailing:
- Steps to reproduce
- Expected vs. actual behavior
- Screenshots (if applicable)

### Suggesting Features
Have an idea? Open an Issue with the "enhancement" label and describe your proposal.

### Submitting Code
1. **Fork the repository** and create your branch from `main`.
2. **Make your changes**. Ensure your code follows the project's style (run `npm run lint`).
3. **Commit your changes**. Write clear, descriptive commit messages.
4. **Push to your branch** and submit a **Pull Request**.
